# Tasks Manager System

A simple Django-based web application for managing daily tasks.  
The system allows users to create, view, edit, complete, and delete tasks.

## Features
- Add new tasks  
- Edit existing tasks  
- View task details  
- Mark tasks as completed  
- Delete tasks  
- Task types: Daily / Weekly / Monthly / Specific date  
- Priority levels: High, Medium, Low  
- Clean UI with responsive layout  

## Technologies Used
- Django 5  
- HTML  
- CSS  
- SQLite (default database)

## How to Run
```
pip install django
python manage.py migrate
python manage.py runserver
```
